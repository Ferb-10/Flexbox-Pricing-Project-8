# Flexbox-Pricing-Project-8

## Overview
This project is a responsive pricing table built using HTML and CSS with Flexbox.
It displays multiple pricing plans in a clean, modern layout that adapts to different screen sizes.
The main goal of this project is to practice Flexbox layout techniques and responsive design.

![image alt](https://github.com/Ferb-10/Flexbox-Pricing-Project-8/blob/4d2e9e91092530476913b9af6c264c0caf24d936/flexbox.png)

## Features
- Pricing table with multiple plans
- Layout built using CSS Flexbox
- Responsive design with media queries
- Clean card-style UI
- Custom typography using Google Fonts

## Technologies Used
- HTML5
- CSS
- Flexbox
- Google Fonts

## What I Learned
- Using Flexbox for horizontal and vertical alignment
- Controlling spacing with gap, justify-content, and align-items
- Building reusable card components
- Creating responsive layouts with media queries
- Designing common UI components used in real-world websites

## 概要
このプロジェクトは、HTMLとCSS（Flexbox）を使用して作成したレスポンシブな料金表ページです。
複数のプランをカード形式で表示し、画面サイズに応じてレイアウトが自動的に切り替わります。
実務でよく使われるUIコンポーネントの実装を目的とした練習プロジェクトです。

## 機能
- 複数プランを表示する料金表
- Flexbox を使ったレイアウト構成
- メディアクエリによるレスポンシブ対応
- カード型のシンプルでモダンなUI
- Google Fonts を使用したタイポグラフィ

## 使用技術
- HTML5
- CSS
- Flexbox
- Google Fonts

## 学んだこと
- Flexboxによる横並び・中央揃えの方法
- gap、justify-content、align-items の使い分け
- 再利用可能なカードコンポーネント設計
- メディアクエリを使ったレイアウト切り替え
- 実務で頻出するUIパターンの実装方法
